# SuveyTool4MBA V2.0 快速上手指南

## 1. 这个系统适合做什么

如果你遇到的问题是：

- 某个管理措施为什么效果不好
- 某种组织或营销策略是否真的有效
- 想把业务直觉转成结构化研究并用数据验证

那么你可以用 SuveyTool4MBA 来完成一套从“业务问题”到“分析解释”的流程。

## 2. 推荐首次使用路径

1. 打开首页
2. 点击 `新建我的项目`
3. 在 `AI 建模助手` 中输入你的业务痛点
4. 复制系统生成的提示词
5. 把提示词粘贴到外部 AI 助手，例如 ChatGPT、DeepSeek、通义千问
6. 把外部 AI 返回的 JSON 粘贴回系统
7. 应用结果，生成你的项目结构
8. 检查并修改研究变量、假设和题项
9. 上传 CSV 数据
10. 执行 `数据体检`
11. 如果有红灯，先剔除拖后腿题项再重算
12. 进入 `数据分析`
13. 在 `研究报告` 步骤中复制提示词到外部 AI，生成更完整的高管风格报告

## 3. 如果你只是想快速理解系统

可以先点击：

- `查看示例项目`

这样可以直接浏览一个完整示例，理解系统整体工作方式。

## 4. 使用时需要注意

- 当前版本不是站内全自动 AI 系统
- 故事转模型与董事会报告都依赖“复制提示词到外部 AI”这一协作方式
- 上传数据时建议使用干净的 CSV 文件
- 如果数据体检出现红灯，先处理题项质量，再继续分析

## 5. 隐私提示

当前版本中：

- 原始业务数据主要在浏览器本地处理
- 系统不会自动把原始数据发送给外部 AI
- 只有你主动复制出去的提示词内容会进入外部 AI 对话
