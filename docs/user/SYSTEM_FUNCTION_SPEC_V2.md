# SuveyTool4MBA V2.0 系统功能说明书

## 1. 系统定位

SuveyTool4MBA V2.0 是一个面向 DBA / MBA 用户的商业决策验证工具。

它不是传统的“统计计算器”，而是一个从业务痛点出发，逐步完成研究设计、数据体检、统计分析与决策解释的工作流系统。

当前版本采用：

- 浏览器本地分析
- 外部 AI 提示词协作

## 2. 目标用户

- 具有丰富业务经验的管理者
- DBA / MBA 学习者
- 需要用量化方法验证商业直觉的用户
- 不希望深度学习统计学术语的非技术用户

## 3. 当前版本主要功能

### 3.1 首页与项目入口

- 支持“新建我的项目”
- 支持“查看示例项目”
- 支持查看研究工作流
- 支持查看未来开发计划书

### 3.2 故事转模型

- 用户输入业务痛点或研究背景
- 系统生成专业提示词
- 用户将提示词复制到外部 AI 助手
- 用户把返回的 JSON 结果粘贴回系统
- 系统自动解析研究标题、研究问题、构念、假设和题项

### 3.3 研究设计工作流

- 项目总览
- 研究问题定义
- 构念与假设管理
- 问卷题项管理

### 3.4 发布与收集

- 问卷状态占位
- 截止日期设置
- 分享链接展示
- 数据收集进度展示

### 3.5 红绿灯数据体检

- 支持上传 CSV 数据
- 本地计算 Cronbach's Alpha
- 本地计算 AVE
- 用红绿灯方式显示数据健康状态
- 标出拖后腿题项
- 支持剔除题项后重新计算

### 3.6 数据分析

- 支持 PLS-SEM 本地分析
- 支持展示路径系数、R²、Bootstrap 等分析结果
- 支持图表和结构路径图展示

### 3.7 董事会报告辅助生成

- 系统根据分析结果整理出适合外部 AI 使用的提示词
- 用户复制提示词到外部 AI 助手
- 外部 AI 生成更适合高管阅读的商业报告文本

## 4. 当前版本边界

当前版本尚未实现：

- 站内 OpenAI / DeepSeek API 自动调用
- 用户 API Key 设置面板
- 自动统计算法路由（T-test / ANOVA / Chi-square）
- 在线问卷收集系统
- 团队协作
- 多组分析
- 中介 / 调节专项分析

## 5. 数据与隐私

当前版本强调本地处理：

- 原始业务数据在浏览器内处理
- 系统不会自动把原始数据上传给外部 AI
- 只有用户主动复制的提示词内容会被带到外部 AI 工具中
